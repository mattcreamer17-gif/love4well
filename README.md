# Love4Well Landing Page

A bespoke, one-off conference page built for Yara at Love4Well (eMerge Americas 2026, stand SMB16). The page mirrors the love4well.com visual language (Gabriela serif, ABeeZee body, Dongle accent; rose, plum, sage, cream palette) and carries a Spanish-first pitch body in her own brand voice. Eduba shows up only in the footer line and in the named case references. The CTA points at Matt Creamer's Calendly.

Intended URL path: /for/love4well-llc

## Files

- `index.html` — single page, no JavaScript.
- `styles.css` — brand tokens from `brand-pack.json`.
- `logo.png` — copied from the brand pack assets folder.

## Notes

- Page language is Spanish, matching the brand. A one-line English translation sits under the headline for any English-first reader who stops by.
- The CTA button and meta link both resolve to `https://calendly.com/thecro-eduba/30min`.
- No pricing, no retainer figures, no fee ranges anywhere on the page.
